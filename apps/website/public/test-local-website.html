CyberEco Website - Local Testing

This file helps you test the exported static website locally before deployment.

Option 1: Use a Simple HTTP Server

Run the following command from the apps/website/out directory:

npx http-server -p 8000

Then open http://localhost:8000

Option 2: Use the Firebase Emulator

Run the following command from the root project directory:

npm run hosting:website

Then open http://localhost:5000

Note: Opening the HTML files directly in your browser may not work correctly due to path issues. Using a web server is recommended.

When tested with a server, the site should display correctly with all styles and JavaScript functionality working.
